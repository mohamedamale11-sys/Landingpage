# MxCrypto SEO Domination (Somali Market)

## Goal
Become the top Somali-language crypto site for high-intent search terms in news, prices, and education.

## Primary keyword clusters
- News intent:
  - `wararka crypto somali`
  - `wararka bitcoin`
  - `wararka ethereum`
  - `wararka memecoin`
  - `bitcoin news somali`
  - `ethereum news somali`
- Price intent:
  - `qiimaha bitcoin maanta`
  - `qiimaha ethereum maanta`
  - `btc price somali`
  - `eth price somali`
- Learn intent:
  - `baro crypto`
  - `free crypto courses`
  - `bitcoin somali`
  - `defi somali`
- Meme/altcoin intent:
  - `memecoin somali`
  - `meme coin somali`

## On-page requirements
- Every target page must have:
  - unique title (contains primary keyword)
  - unique meta description (human-readable, Somali-first)
  - canonical URL
  - at least one H1
  - internal links to 2+ related pages
  - structured data (`ItemList`, `FAQPage`, or `NewsArticle` where relevant)

## Technical SEO requirements
- Keep `robots.txt` + `sitemap.xml` + `news-sitemap.xml` valid.
- Include only indexable pages in sitemap.
- Keep page speed high:
  - cached API responses
  - reduced layout shifts
  - eager load only hero image
- Ensure mobile-first rendering quality.
- Add WebSite `SearchAction` structured data.
- Keep internal linking dense between all topic pages.

## Publishing cadence
- News ingestion every 15 minutes.
- Add at least 2 new “evergreen” Somali explainers weekly.
- Refresh top price/news pages daily with updated copy snippets and internal links.

## 90-day execution plan
- Phase 1 (Week 1-2): indexation + crawl
  - submit both sitemaps in Search Console
  - inspect and request indexing for all topic pages
  - fix any crawl/index errors within 24h
- Phase 2 (Week 3-6): topical authority
  - publish 3 articles/week targeting long-tail Somali keywords
  - interlink each new article to 2 topic pages + 1 evergreen page
  - optimize title/description based on real impressions
- Phase 3 (Week 7-12): dominance
  - build backlinks from Somali finance/tech communities
  - push branded searches (`MxCrypto`, `MxCrypto Somali`) via social
  - run weekly content pruning/merge for thin or duplicate pages

## Authority and trust (E-E-A-T)
- Keep clear disclaimer/privacy/terms pages live.
- Add author/editor bylines where possible.
- Add source verification workflow internally before publish.
- Publish editorial policy page for transparency.

## Off-page growth requirements
- Get backlinks from:
  - Somali finance blogs
  - Somali tech communities
  - crypto education channels
- Share each new pillar page on Telegram, X, and WhatsApp communities.

## KPI targets (first 90 days)
- Rank top 3 for:
  - `wararka crypto somali`
  - `qiimaha bitcoin maanta`
  - `wararka bitcoin`
  - `memecoin somali`
- CTR from Search > 6%
- Organic sessions +15% week-over-week for first 8 weeks
